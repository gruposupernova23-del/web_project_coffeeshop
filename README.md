# Coffeeshop

## Descripción del proyecto

Página web de una cafetería llamada Coffeeshop. Incluye encabezado con navegación, sección de recetas con videos embebidos y un formulario para reservar mesa.

## Tecnologías y técnicas utilizadas

- HTML5 semántico
- CSS3
- Metodología BEM
- Estructura de archivos BEM (CSS dividido por bloques)
- Normalize.css
- Google Fonts
- Iframes (videos de YouTube)

## Planes de mejora

- Hacer el formulario funcional con JavaScript (validación y envío real).
- Agregar versión responsive para pantallas pequeñas.
- Mejorar accesibilidad (mejor contraste, focus visible, aria-label donde aplique).
